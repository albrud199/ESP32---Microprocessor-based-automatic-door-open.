# ESP32---Microprocessor-based-automatic-door-open.
This project implements a smart, automated door opener system using an ESP32 microprocessor (a powerful microcontroller) as the central brain. The system is designed to detect when a person is near and automatically open the door for them.


Designed an IoT-enabled automatic door opener using ESP32, ultrasonic sensing, and servo motor control — automatically opening at close proximity and allowing remote trigger via Firebase cloud database over Wi-Fi. Demonstrates embedded C/C++ programming, real-time sensor integration, and secure cloud communication.
